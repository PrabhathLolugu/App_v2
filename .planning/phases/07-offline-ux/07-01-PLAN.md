**connectivity_event.dart:**
- Use @freezed sealed class with two events: checkConnectivity(), connectivityChanged(InternetStatus status)
- Part directive: 'connectivity_event.freezed.dart'

**connectivity_state.dart:**
- Use @freezed sealed class with two states: online(), offline()
- Part directive: 'connectivity_state.freezed.dart'

**connectivity_bloc.dart:**
- @injectable annotation for DI registration
- Extends Bloc&lt;ConnectivityEvent, ConnectivityState&gt;
- Constructor takes optional InternetConnection parameter (for testing), defaults to InternetConnection()
- Initialize with ConnectivityState.online() (optimistic default prevents flash on app start)
- Subscribe to _internetConnection.onStatusChange in constructor
- **Critical debouncing logic:** For InternetStatus.disconnected, use Timer(Duration(seconds: 3)) before emitting offline state. For connected, emit immediately.
- Cancel _debounceTimer before starting new one
- Register event handlers: on&lt;CheckConnectivityEvent&gt;, on&lt;ConnectivityChangedEvent&gt;
- Override close() to cancel subscriptions and timer: _subscription?.cancel(), _debounceTimer?.cancel()
- Use hasInternetAccess in CheckConnectivityEvent handler to get current status

Import: internet_connection_checker_plus, dart:async (for Timer), flutter_bloc, injectable, freezed_annotation.

Create OfflineBanner widget that listens to ConnectivityBloc and shows/hides MaterialBanner.

**Structure:**
- StatelessWidget that returns BlocConsumer&lt;ConnectivityBloc, ConnectivityState&gt;
- listenWhen: only trigger when state type changes (previous.runtimeType != current.runtimeType) to avoid duplicate banner shows
- listener:
  - When ConnectivityOffline: ScaffoldMessenger.of(context).showMaterialBanner() with content "No internet connection", backgroundColor: Theme.of(context).colorScheme.error, leading: Icon(Icons.wifi_off, color: Colors.white), actions: [const SizedBox.shrink()] (required by MaterialBanner API)
  - When ConnectivityOnline: ScaffoldMessenger.of(context).hideCurrentMaterialBanner(), then show SnackBar with "Back online", duration: 2 seconds, backgroundColor: Colors.green
- builder: return const SizedBox.shrink() (invisible widget, all logic in listener)

Import: flutter/material, flutter_bloc, connectivity_bloc/connectivity_state.
