   - @Supabase(name: 'metadata') final Map<String, dynamic>? metadata (for tags)

   - Extract tags from metadata['tags'] as List<String>
   - Pass author?.toDomain() for authorUser field
   - Map all fields according to domain entity structure

5. Add fromDomain() factory constructor:
   - Accept ImagePost entity parameter
   - Convert tags to metadata map: {'tags': post.tags}
   - Convert authorUser to UserModel.fromDomain() if not null
   - Map all fields from entity to model
